# outing-shiori

## プロジェクト概要
「日程・予算・出発地・旅スタイルを入力するとAIが旅行プランを提案し、
タイムライン形式のしおりとして表示・URLで共有できるWebアプリ」

## 技術スタック
- React 19 + TypeScript + Vite 7
- Tailwind CSS v4
- AI提案: Claude API
- 共有機能: URLエンコード or Supabase（未定）
- デプロイ: Vercel（予定）

## 開発計画
1. 条件入力フォーム + Claude APIでプラン提案
2. タイムライン形式のしおりUI
3. 地図連携（スポットのピン表示）
4. URLによる共有機能
5. デザイン仕上げ
